<b>Step1</b> create a virtualenvironment :
```virtualenv client```

<b>Step2</b> activate virtual environment:
```client/Scripts/Activate```

<b>Step3</b> Install all requiremnts package:

```pip install -r requirements.txt ```

<b>Step4</b> run server:
```python server.py```
